NEXA is a lightweight Chrome extension that helps users assess clothing size risk before purchasing online.

## Problem
Online fashion shopping suffers from high return rates due to sizing uncertainty.

## Solution
NEXA provides a simple, privacy-first size risk signal based on user body inputs.

## Why Extension First
- Runs at buying moment
- Zero friction
- No backend required

## Status
Early-stage prototype. Infrastructure-first approach.
